# Jiangtherapee SRHQ beta

Jiangtherapee SRHQ beta is a local Windows beta build for SRHQ burst super-resolution processing.

## Download

Download the latest beta from the GitHub Releases page:

https://github.com/y-g-jiang/Jiangtherapee-SRHQ-beta/releases/latest

Recommended asset:

- `Jiangtherapee-SRHQ-beta-Setup.exe`: self-extracting installer for normal users.
- `SRHQ_Release_beta_watermarked.zip`: portable package for manual extraction.

## Notes

- This is a beta build.
- Output images include a beta watermark.
- The formal version is planned for mid-July at https://y-g-jiang.github.io.
